Detailed Description

Applies gamma correction to an image.

Performs the operation: output = 255 * (input/255)^gamma

  • Gamma < 1.0: Brightens dark regions (e.g., gamma=0.5)
  • Gamma = 1.0: No change
  • Gamma > 1.0: Darkens bright regions (e.g., gamma=2.0)

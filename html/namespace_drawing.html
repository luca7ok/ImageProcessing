Detailed Description

Provides 2D primitive drawing functions for images.

Implements common rasterization algorithms

  • Midpoint circle algorithm
  • Bresenham's line algorithm
  • Rectangle outline drawing
    Note
    All functions perform bounds checking and modify the image in-place.

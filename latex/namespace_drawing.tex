\doxysubsection{Detailed Description}
Provides 2D primitive drawing functions for images. 

Implements common rasterization algorithms
\begin{DoxyItemize}
\item Midpoint circle algorithm
\item Bresenham\textquotesingle{}s line algorithm
\item \doxylink{class_rectangle}{Rectangle} outline drawing \begin{DoxyNote}{Note}
All functions perform bounds checking and modify the image in-\/place. 
\end{DoxyNote}

\end{DoxyItemize}

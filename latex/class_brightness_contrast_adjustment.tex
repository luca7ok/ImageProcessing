\doxysubsection{Detailed Description}
Adjusts image brightness and contrast using linear transformation. 

Applies the formula\+: {\ttfamily output = alpha \texorpdfstring{$\ast$}{*} input + beta}, where\+:
\begin{DoxyItemize}
\item {\ttfamily alpha} controls contrast (1.\+0 = no change, \texorpdfstring{$<$}{<}1.\+0 = reduced contrast, \texorpdfstring{$>$}{>}1.\+0 = increased contrast)
\item {\ttfamily beta} controls brightness (0 = no change, positive = brighter, negative = darker) 
\end{DoxyItemize}

\doxysubsection{Detailed Description}
Applies gamma correction to an image. 

Performs the operation\+: {\ttfamily output = 255 \texorpdfstring{$\ast$}{*} (input/255)\texorpdfstring{$^\wedge$}{\string^}gamma}
\begin{DoxyItemize}
\item Gamma \texorpdfstring{$<$}{<} 1.\+0\+: Brightens dark regions (e.\+g., gamma=0.\+5)
\item Gamma = 1.\+0\+: No change
\item Gamma \texorpdfstring{$>$}{>} 1.\+0\+: Darkens bright regions (e.\+g., gamma=2.\+0) 
\end{DoxyItemize}
